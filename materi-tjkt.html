Command Prompt (CMD)

Apa itu Command Prompt?

Command Prompt (CMD) adalah antarmuka baris perintah (CLI) bawaan Windows yang memungkinkan pengguna untuk berinteraksi dengan sistem operasi menggunakan perintah teks. CMD sangat berguna untuk administrasi jaringan, troubleshooting, dan manajemen file.

# Perintah Dasar Sistem
C:\> dir           // tampilkan isi folder
C:\> cd nama_folder // pindah direktori
C:\> mkdir folder  // buat folder baru
C:\> del file.txt  // hapus file
C:\> cls           // bersihkan layar
C:\> exit          // tutup CMD
# Perintah Jaringan
C:\> ipconfig
Ethernet adapter Local Area Connection:
   IPv4 Address: 192.168.1.10
   Subnet Mask:  255.255.255.0
   Gateway:      192.168.1.1

C:\> ping 8.8.8.8
Reply from 8.8.8.8: bytes=32 time=12ms TTL=115

C:\> tracert google.com
Tracing route to google.com [172.217.0.46]

C:\> netstat -an   // lihat koneksi aktif
C:\> nslookup google.com // DNS lookup
C:\> arp -a        // lihat tabel ARP
# Manajemen File & Sistem
C:\> copy a.txt b.txt   // salin file
C:\> move a.txt C:\    // pindah file
C:\> ren lama.txt baru.txt // rename
C:\> type file.txt    // tampilkan isi file
C:\> tree             // struktur folder

# Perintah Sistem
C:\> systeminfo       // info sistem
C:\> tasklist         // proses berjalan
C:\> taskkill /PID 1234 // hentikan proses
C:\> shutdown /s /t 0 // matikan PC
C:\> shutdown /r /t 0 // restart PC
# Perintah Lanjutan Jaringan
C:\> netsh wlan show profiles
Profiles on interface Wi-Fi:
  Group policy profiles: None
  User profiles: MyWiFi

C:\> netsh interface ip set address
C:\> route print     // tabel routing
C:\> pathping 8.8.8.8 // path & loss

IP Address

Apa itu IP Address?

IP Address (Internet Protocol Address) adalah alamat numerik unik yang diberikan kepada setiap perangkat dalam jaringan komputer. IP Address berfungsi sebagai identitas perangkat untuk dapat berkomunikasi satu sama lain dalam jaringan lokal maupun internet.

Struktur IPv4

IPv4 terdiri dari 32 bit yang dibagi menjadi 4 oktet (masing-masing 8 bit), ditulis dalam format desimal:

IPv6 — Generasi Berikutnya

IPv6 menggunakan 128-bit address, ditulis dalam 8 grup hexadecimal 16-bit:

# Format IPv6
2001:0db8:85a3:0000:0000:8a2e:0370:7334

# Dapat disingkat (hapus leading zeros & "::")
2001:db8:85a3::8a2e:370:7334

# Loopback IPv6
::1  ← setara dengan 127.0.0.1

# Total alamat IPv6
340 undecillion (3.4 × 10^38)

Voltase & Watt Perangkat Elektronik

Hukum Ohm & Daya Listrik

# Rumus Dasar Listrik

V = I × R       // Hukum Ohm
P = V × I       // Daya (Watt)
P = I² × R      // Daya via Resistansi
P = V² / R      // Daya via Voltase
E = P × t       // Energi (kWh)

# Keterangan Satuan
V = Volt (Tegangan)
I = Ampere (Arus)
R = Ohm (Resistansi)
P = Watt (Daya)
t = Jam (Waktu)

Aplikasi Virtual

Apa itu Virtualisasi?

Virtualisasi adalah teknologi yang memungkinkan pembuatan versi virtual dari perangkat keras, sistem operasi, atau jaringan. Dengan virtualisasi, satu komputer fisik dapat menjalankan beberapa sistem operasi secara bersamaan melalui lapisan software yang disebut Hypervisor.

🔵

VirtualBox

Hypervisor Type-2 open-source dari Oracle. Gratis dan mendukung berbagai guest OS. Ideal untuk pelajar dan lab jaringan.

GRATIS OPEN SOURCE WINDOWS/MAC/LINUX
🟠

VMware Workstation

Hypervisor Type-2 profesional dari VMware. Performa tinggi, fitur lengkap untuk enterprise. Versi Player tersedia gratis.

BERBAYAR ENTERPRISE HIGH PERFORMANCE
🟣

Hyper-V

Hypervisor Type-1 bawaan Windows 10/11 Pro dan Server. Digunakan untuk server production dan infrastruktur enterprise Microsoft.

TYPE-1 BUILT-IN WINDOWS ENTERPRISE
🟦

Cisco Packet Tracer

Simulator jaringan dari Cisco untuk belajar konfigurasi router, switch, dan protokol jaringan tanpa hardware fisik.

SIMULATOR CISCO GRATIS UNTUK PELAJAR
🟡

GNS3

Graphical Network Simulator yang lebih advanced dari Packet Tracer. Mendukung image router/firewall nyata dan integrasi dengan VM.

SIMULATOR OPEN SOURCE ADVANCED
🐧

Proxmox VE

Platform virtualisasi berbasis Linux yang menggabungkan KVM dan LXC. Digunakan untuk home lab dan production server.

TYPE-1 OPEN SOURCE HOME LAB

Jenis Hypervisor

# Type-1 (Bare-Metal Hypervisor)
Berjalan langsung di hardware
Hardware → Hypervisor → VMs

# Contoh:
• VMware ESXi
• Microsoft Hyper-V
• Proxmox VE
• Xen Server

# Kelebihan:
+ Performa tinggi
+ Efisiensi resource
+ Cocok production
# Type-2 (Hosted Hypervisor)
Berjalan di atas OS host
Hardware → OS Host → Hypervisor → VMs

# Contoh:
• VirtualBox
• VMware Workstation
• Parallels Desktop
• QEMU

# Kelebihan:
+ Mudah diinstall
+ Cocok untuk belajar
+ Berjalan di atas OS biasa
